# SQL Practice Tutor

サイバーエージェント データ分析職の面接対策用SQL練習アプリケーション

## 概要

このアプリは**答えを直接教えない**学習支援ツールです。段階的なヒント、SQL構文リファレンス、スキーマ参照を通じて、自力でSQLを書く力を養います。

## 機能

- **練習問題一覧**: 難易度・カテゴリ別にフィルタリング可能な10問の練習問題
- **SQLエディタ**: シンタックスハイライト付きのコードエディタ
- **段階的ヒント**: テーブル情報 → 使用句 → クエリ構造の3段階ヒント
- **スキーマ参照**: テーブル構造とサンプルデータの確認
- **SQL構文リファレンス**: 主要なSQL構文の説明と使用例
- **学習進捗管理**: 完了した問題の追跡とローカルストレージへの保存

## 技術スタック

- React 18 + TypeScript
- Vite
- Tailwind CSS v4
- CodeMirror 6 (SQLエディタ)
- React Router v6
- shadcn/ui コンポーネント

## 開発

```bash

# プレビュー
npm run preview
```

## データ構造

### テーブル

- `users`: ユーザー情報（流入元、デバイス、年齢層、地域）
- `payments`: 決済情報（金額、通貨、ステータス）
- `events`: ユーザーイベント（アクション、タイムスタンプ）
- `ad_costs`: 広告費用（ソース、キャンペーン、費用）

### 問題カテゴリ

- **基本クエリ**: SELECT, WHERE, ORDER BY
- **集計**: GROUP BY, COUNT, SUM, AVG
- **分析**: ウィンドウ関数, サブクエリ
- **セグメント分析**: CASE WHEN, コホート分析
